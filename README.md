# JobSynk - 智能简历自动投递系统

## 项目简介

JobSynk 是一个基于 Go 语言开发的自动化求职助手，通过智能技术简化求职流程。系统使用 Playwright 自动化爬取招聘网站岗位信息，集成大语言模型对岗位进行智能评分和排序，基于您的简历内容自动生成个性化招呼语，并实现一键自动投递。

核心功能包括多维度岗位评分、智能招呼语生成、自动化投递执行和状态跟踪管理。系统能够智能筛选优质岗位，避免低质量机会，提高求职效率和成功率。所有数据本地存储，保障隐私安全。
